# Concepts

Concept guides explain the VTRACE mental model. They are user-facing docs, not
controlled specifications.

| Guide | Use It When |
|---|---|
| `vtrace-in-one-page.md` | You need the shortest useful explanation. |
| `v-shape-for-software.md` | You need to understand left-side and right-side stages. |
| `specs-vs-docs.md` | You need to separate controlled specs from user docs. |
| `work-packages-and-evidence.md` | You need to understand procedural execution. |
