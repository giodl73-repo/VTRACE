# Tutorial 01: Minimum Slice

Goal: create the smallest useful VTRACE package.

1. Create `docs/vtrace/MISSION.md`.
2. Add one `REQ-*` row in `REQUIREMENTS.md`.
3. Add one `SPEC-*` row in `SPECIFICATION_BASELINE.md`.
4. Link the requirement, spec, work package, and evidence in `TRACE.md`.
5. Add `VERIFICATION.md` and `REVIEW.md`.
6. Run `vtrace validate <repo>`.

Expected output:

```text
VTRACE validation passed
```

If validation fails, fix the file and line findings before adding more process
depth.
