# VTRACE Agent Instructions

Read `README.md`, `PRODUCT_PLAN.md`, and `context/waves/PHASES.md` before
changing this repo.

VTRACE adapts public NASA systems engineering guidance for software codebases.
Keep the distinction clear: this repo is NASA-inspired, not NASA-authored or
NASA-endorsed.

Prefer small, auditable changes. Every new framework artifact should name:

- the engineering decision it supports,
- the trace links it expects,
- the evidence it produces,
- the validation command that checks it.
