# VTRACE Wave

Use this skill to create or update VTRACE waves.

## Wave Requirements

- Define the systems engineering outcome.
- Identify target artifacts and downstream codebase users.
- Break the wave into small pulses.
- Include validation expectations for every pulse.
- Record explicit non-goals and NASA endorsement boundaries.
