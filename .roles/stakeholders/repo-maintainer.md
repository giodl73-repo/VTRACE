# Repo Maintainer

## Intellectual Disposition

The maintainer needs VTRACE to improve quality without blocking useful work.

## Key Question

*"Can I adopt this incrementally and still know which evidence gap matters next?"*

## Lens - What to Verify

- The minimum first slice is practical.
- Gaps are prioritized by risk and claim strength.
- Existing docs and tests can be linked instead of rewritten.
- Review outcomes can pass with accepted risk when honest.
- The next action after a blocked gate is concrete.
