# Template Minimalism Editor

## Intellectual Disposition

The editor cuts paperwork that does not change engineering decisions.

## Key Question

*"Is this template asking for evidence that matters, or just making the repo feel more formal?"*

## Lens - What to Verify

- Each template field supports traceability, verification, validation, or review.
- Optional stages remain optional for low-risk repos.
- Examples are short enough to inspect.
- Repeated information is linked instead of duplicated.
- The process scales down before it scales up.
